# Monopoly Game Arena

A deterministic, fully-tested Monopoly rules engine with:

- Standard board, spaces, rents, and decks (Chance & Community Chest)
- Turn flow, doubles, jail, taxes, and passing GO
- Property purchases, auctions, building (even-build), mortgages
- Trading (properties, cash, jail cards) with validation
- Clear event logging, snapshots, and optional web/DB integration
